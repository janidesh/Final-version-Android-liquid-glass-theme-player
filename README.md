**Apex Crystal** is a high-performance, ultra-modern Media Engine built with Jetpack Compose. It redefines the Android UI experience with a **Liquid Purple Glass** aesthetic inspired by iOS Acrylic and Windows Fluent design systems.

---

## ✨ Key Features

### 🎵 Audio Master Experience
- **Liquid Purple UI:** A full-screen immersive glass-morphism interface with high-intensity Gaussian blur.
- **Floating Glass Disc:** A real-time rotating disc with GPU-accelerated shadows.
- **5-Band Equalizer:** Hardware-linked EQ for professional-grade sound tailoring.
- **A-B Repeat:** Seamlessly loop specific sections of your favorite tracks.

### 🎬 Cinema Video Player
- **Orientation Control:** One-tap toggle between Portrait and Landscape (Cinema Mode).
- **Edge-to-Edge Visuals:** Minimalist controls that stay out of the way of your content.
- **Subtitle Support:** Integrated logic for external subtitle injection.

### 🌊 Next-Gen Design
- **Real-Time Liquid Backgrounds:** Dynamic neon "blobs" that shift and flow behind a 110.dp blur.
- **Glass-Morphism:** Every card and button uses alpha-blending and linear gradients for a realistic glass feel.
- **Neon Branding:** Custom "Janith Rathnayake Creations" signature with active glow effects.

## 🚀 Animations Gallery

> [!TIP]
> This app uses the **Compose Animation API** to achieve liquid-smooth transitions.

* **Splash Screen:** Neon pulse and scale-in animation.
* **Media List:** Animated visibility with `expandVertically` and `fadeIn`.
* **Disc Rotation:** `InfiniteTransition` with `LinearEasing` for a perfect 360° spin.
* **Liquid Flow:** Background canvas circles moving on a `RepeatMode.Reverse` loop.
